## 项目简介

Cloud Save Max 是基于 [cloud-auto-save-x](https://github.com/OzoO0/cloud-auto-save-x) 改造的网盘自动转存工具，保留了原项目的所有功能，并新增了以下特性：

### 新增功能

- 🎯 **分享者黑白名单**：支持优选和屏蔽特定分享者
- 🔍 **搜索筛选/过滤词**：搜索结果支持关键词筛选和过滤
- 📁 **文件夹筛选/过滤**：支持按文件夹名称筛选或过滤
- 📄 **文件关键词筛选/过滤**：支持按文件名关键词筛选或过滤
- 📅 **时间过滤**：支持按文件/文件夹时间过滤
- 🏷️ **任务模板**：支持保存和应用任务配置模板
- 🔄 **自动换链增强**：链接失效时自动触发换链
- 📊 **缓存优化**：搜索和验证结果缓存
- 🎬 **追剧面板增强**：支持直接创建同步任务
- 📝 **魔法变量扩展**：新增 `{E0}`、`{E2}` 变量
- 🔧 **预览窗优化**：更直观的文件筛选标记和操作
- ⚙️ **系统设置重构**：更清晰的设置页面布局
